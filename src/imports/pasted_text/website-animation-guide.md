Design and implement a world-class enterprise website animation system for Exceptional Solutions.

The website serves enterprise CIOs, IT Directors, Integration Architects, Operations Leaders, and Technology Executives.

The experience must feel premium, modern, trustworthy, intelligent, and highly polished.

Reference quality:
Stripe
Linear
Vercel
Retool
Ramp
Mercury
Firecrawl
Palantir
MuleSoft
Datadog

Avoid:
Cheap startup animations
Bounce effects
Cartoon effects
Overly playful interactions
Generic fade-ins everywhere

Design principles:

1. Every animation must have purpose.
2. Motion should communicate hierarchy and relationships.
3. Motion should guide attention.
4. Motion should feel engineered, not decorative.
5. Use subtle sophistication over flashy movement.
6. Maintain 60fps performance.
7. Respect prefers-reduced-motion.

----------------------------------------------------
GLOBAL MOTION SYSTEM
----------------------------------------------------

Use Framer Motion.

Animation timing:

Fast:
150ms - 250ms

Standard:
300ms - 450ms

Complex:
600ms - 800ms

Easing:
easeOut
easeInOut
cubic-bezier(0.16, 1, 0.3, 1)

Never use:
Bounce
Elastic
Spring overshoot
